# @mdpress/plugin-google-analytics

Google analytics plugin

# Install

yarn add -D @mdpress/plugin-google-analytics
# OR npm install -D @mdpress/plugin-google-analytics

WARNING

We currently recommend using Yarn instead of npm to install all dependencies if you are using Google Analytics Plugin, because npm fails to generate the correct dependency tree in this case.

# Usage

module.exports = {
  plugins: [
    [
      '@mdpress/google-analytics',
      {
        'ga': '' // UA-00000000-0
      }
    ]
  ]
}

TIP

Please be aware of GDPR (2018 reform of EU data protection rules) as IPs are anonymized automatically.

# Options

# ga

  • Type: string
  • Default: undefined

Provide the Google Analytics ID to enable integration.
